### About ###

Create professional backend software with [JVx](https://www.sibvisions.com/jvx) and use your domain model which is already available in your Java EE based web application. With JVx EE you can use the configured EntityManager and JPA implementation of your web application. Reuse your existing DAOs or EAOs without changes in your application.

Features

* Use existing domain models
* Use existing DAOs and EAOs
* Use JPA for your JVx application
